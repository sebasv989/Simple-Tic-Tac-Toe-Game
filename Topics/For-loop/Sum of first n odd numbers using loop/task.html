Sum of first n odd numbers using loop

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a program that reads an integer n and prints the sum of the first n positive odd numbers using a for loop. The input consists of a single integer n (1 <= n <= 100) on a line by itself. The output should be the sum of the first n positive odd numbers printed on a single line.
Sample Input:
5

Sample Output:
25


Sample Input:
10

Sample Output:
100



Memory limit: 256 MB
Time limit: 5 seconds
